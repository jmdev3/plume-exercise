## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project structure

```
plume-exercise/
 ├─ src/
 │   ├─ app/                 # Simple App Router
 │   │   ├─ layout.tsx
 │   │   ├─ page.tsx         # home page
 │   ├─ components/          # UI components
 │   ├─ hooks/               # custom hooks
 │   ├─ lib/                 # clients & helpers
 │   ├─ types/               # types
 |   └─ providers/           # custom providers
```
